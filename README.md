# Drill app

This app was developed with Python **tkinter** library. The main goal of this app is to help students to prepear for their exams. In order to use this app
please download this repository and run `MainWindow.pyw` file. Then create your own text file with questions and answers and load it 
to the app. You can find out how exactly the text file should look like by clicking *Info* in top-right corner of the app.
